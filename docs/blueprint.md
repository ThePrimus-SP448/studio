# **App Name**: StockWise

## Core Features:

- User Authentication: Secure user login and signup using Supabase.
- Live Stock Data: Fetch and display real-time stock market data from an external API.
- Virtual Trading: Allow users to buy and sell stocks virtually with simulated funds.
- Portfolio Management: Track user's virtual stock holdings, purchase history, and current positions.
- Profit/Loss Tracking: Calculate and display profit and loss for each stock and the overall portfolio.
- Interactive Charts: Visualize stock price trends and portfolio performance with interactive charts.
- Personalized Investment Tool: AI powered investment tool providing tailored recommendations to enhance the trading strategy. It acts as a financial summarization and educational assistant for each user, reasoning about their goals to offer helpful advice, incorporating real time updates if relevant.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) for trust and stability.
- Background color: Very light blue-gray (#E8EAF6) for a clean and modern feel.
- Accent color: Vivid cerulean (#03A9F4) to highlight key information and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral look.
- Use clean, minimalist icons to represent stock trends and portfolio actions.
- Design a clean and intuitive layout with clear sections for the dashboard, portfolio, and trading interface.
- Implement subtle animations for transitions and data updates to enhance user experience.
